# Development
1. Clone repository.
2. Run `npm install`.
IN THIS ORDER
USE THREE DIFFERENT CONSOLE TABS
3. Run `redis-server`.
4. Run `npm start`.
5. Run `npm run watch`.
6. IF you are working on client side you need to Run `npm watch` & `npm start`.
